# Minecraft Monster Music Pack
 Changes Minecraft's background music into Monster songs.

Where to download:
Click the most recent version in Releases on the right side of the screen.

How to install:
1. Go into Minecraft
2. Go to Options > Resource Packs > Open Pack Folder. This will open your resource pack folder
3. Drag the entire 'Monster_Music_Pack.zip' file into the resource pack folder
4. Enjoy songs such as sonder, hello this is an intro, and of course EEEEEEGAD!!!!!!!!
